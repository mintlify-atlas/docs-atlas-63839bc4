## What you can do

- **Evaluate** LLM outputs using the `@evaluator()` decorator or managed `RemoteEvaluator` — supports boolean, float, string, and structured `EvaluationResult` return types
- **Experiment** systematically by running tasks over datasets and collecting evaluation results with `run_experiment()`
- **Trace** any Python function or code block using `@traced()` and `start_span()`, built on OpenTelemetry
- **Manage prompts** with versioning, labels, and multiple template engines (f-string, Mustache, Jinja2)
- **Integrate** with OpenAI, Anthropic, and other providers through OpenInference instrumentors

## Get started

<Steps>
  <Step title="Install the SDK">
    ```bash
    pip install patronus
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export PATRONUS_API_KEY="your-api-key"
    ```
    Or create a `patronus.yaml` config file in your project directory.
  </Step>
  <Step title="Initialize and evaluate">
    ```python
    import patronus
    from patronus.evals import RemoteEvaluator

    patronus.init()

    evaluator = RemoteEvaluator("judge", "patronus:is-concise")
    result = evaluator.evaluate(
        task_input="Summarize the report",
        task_output="The report shows a 12% revenue increase in Q3."
    )
    print(f"Passed: {result.pass_}, Score: {result.score}")
    ```
  </Step>
  <Step title="Explore the docs">
    Continue to the [Quickstart](/getting-started/quickstart) for more examples covering tracing, experiments, and custom evaluators.
  </Step>
</Steps>
